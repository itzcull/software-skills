# Design Patterns Overview

## Introduction

Design patterns are "common approaches to solving similar problems." The seminal work on design patterns is the 1995 book "Design Patterns" by the Gang of Four (GoF), which originally described 23 patterns.

## Alphabetical List of Design Patterns

### Notable Patterns
- Abstract Factory
- Adapter
- Builder
- Chain of Responsibility
- Decorator
- Facade
- Factory Method
- Mediator
- Memento
- Null Object
- Observer
- Proxy
- Repository
- Singleton (often considered an antipattern)
- Specification
- Strategy
- Strangler Fig
